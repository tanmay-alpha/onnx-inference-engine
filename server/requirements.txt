# Pinned to the versions used during initial development on
# Python 3.11. Pinning matters here because (a) FastAPI has
# shipped breaking changes between minor versions, and (b) torch

# a different interpreter will fail at install time.
#
# `crucible_py` (the C++ pybind11 extension) is built separately
# via `cmake --build build/release` and added to PYTHONPATH at
# runtime. It is NOT in this file because pip can't build it
# from a wheel.
